# YSC-SO-M04-4 脉冲激光驱动器（综合型）说明书

> 小型 OEM 脉冲激光驱动模块，通过 TTL 外触发或内部时钟控制激光器，Modbus RTU 串口配置参数。

## 关键信息

### 主要规格

| 参数 | 值 |
|------|----|
| 重复频率 | 1 Hz – 1 MHz（默认 5 kHz） |
| 脉冲宽度 | 5 – 200 ns（默认 10 ns） |
| 驱动电压 | 5 V（V 端）~ 30 V（可调） |
| 驱动电流 | 0 – 80 A |
| 控制电源 | DC 5 V ± 0.1 V |
| 触发信号 | TTL |
| 尺寸 / 重量 | 40 × 22 mm / ~20 g |

### 接口引脚

**P3**：TTL 外触发 / 内部同步信号（二选一）

**P4**（串口控制）：

| 引脚 | 说明 |
|------|------|
| GND | 接地 |
| RX  | 串行数据输入（激光器接收） |
| TX  | 串行数据输出（激光器发送） |
| 5V  | 电源（激光器输出，可为外部供参考） |

**P1**：主电源（+ / −）

### 通信协议

- **Modbus RTU，9600 bps，8N1**
- 帧格式：`[func 1B][data 4B big-endian][CRC16 2B little-endian]` = 7 字节

| 功能码 | 功能 | data 说明 |
|--------|------|-----------|
| `0x01` | 设置电压等级 | 1–200（值越大输出越高） |
| `0x02` | 设置频率 | Hz，范围 1–1000000 |
| `0x03` | 设置脉宽 | 取 5 的倍数（5ns 单位）：1=5ns, 2=10ns … 40=200ns |
| `0x04` | 设置触发模式 | 0=关闭，1=内部触发，2=**外触发**（PF32 sys_master 必选） |
| `0x06` | 读取当前参数 | data=`0x00000000`；响应约 13–14 字节（字段映射待实物核实） |

响应规则：成功时响应帧与发送帧相同（echo-back）。读参数返回多字节。

### 本项目接线方案

- **P3**（TTL 外触发）← PF32 TRIG 直接连接（同轴，无需电平转换）
- **P4 RX** ← 哪吒 CN3 pin 8（TX，3.3V）：5V TTL VIH = 2.0V，3.3V 可驱动，**无需升压**
- **P4 TX** → 分压 → 哪吒 CN3 pin 10（RX，3.3V）：需降压保护 GPIO

分压电路（P4 TX → 哪吒 RX）：
```
激光 P4 TX ──[10kΩ]──┬── 哪吒 CN3 pin 10
                     [20kΩ]
                      GND
```

- **P1**：独立 5V 适配器供电

### 注意事项

- 外触发模式下，激光重复频率由 PF32 TRIG 决定，`setFreqHz` 无效
- 使用前须专业培训，禁止直视激光
- TX、RX 需按方案连接；5V 和 GND 需接硬件电路中的分隔保护线路
