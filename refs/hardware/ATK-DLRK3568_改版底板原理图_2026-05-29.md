# ATK-DLRK3568_改版底板原理图_2026-05-29

> ATK-DLRK3568 RK3568 开发板改版底板原理图（相对 V1.5 版），STM32 + TMC2209 电机驱动板上集成。

## 关键信息

### 改版相对 V1.5 的主要变化

- **新增**：STM32F103C8T6 + TMC2209 ×2 **板上焊死**（不再外挂模块）
- **新增**：RS422 接口
- **删除**：RS232、CAN、Audio、HDMI、TF 卡、IR、6 轴传感器、HW_ID、KEY 矩阵

### UART 跳线映射（权威，见 `docs/board_custom_uart_mapping.md`）

| 跳线 | 短接方式 | 对应节点 | 连接目标 |
|------|---------|---------|---------|
| JP2 | 3-5 / 4-6 短接 | `/dev/ttyS4`（UART4） | STM32 USART1（电机控制） |
| JP3 | 1-3 / 2-4 短接 | `/dev/ttyS3`（UART3） | RS422 收发器（默认空置） |

### 电机控制链路

```
RK3568 /dev/ttyS4 → JP2 → STM32F103C8T6 USART1 → TMC2209×2 → MOTOR1/MOTOR2
```

- 协议：19200 8N1
- MOTOR1：调焦直线滑台
- MOTOR2：光圈齿轮

### 激光备用方案（非默认）

- 拔 JP3 跳线 → `/dev/ttyS3` 引脚直接引出 → 可接激光器串口
- **默认方案**：激光接哪吒 CN3（PF32 就近 + 闭环延迟最低）；3568 ttyS3 接激光为临时调试备案
